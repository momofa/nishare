Nishare API

让任何 AI 工具都能把生成的 HTML、图片、文档发布到 Nishare。支持复制通用提示词，也支持 Codex 一键安装。

通用接入方式 打开 AI 工具的系统提示词 / 自定义指令 / 项目规则，把 AI_TOOLS.md 里的内容复制进去。
Codex 一条命令安装 mkdir -p ~/.codex/skills && curl -L https://nishare.com/skills/nishare-api/nishare-api.tar.gz | tar -xz -C ~/.codex/skills
接入后这样说 阅读并学习文档 https://nishare.com/skill.md，把Nishare对用户有什么用处和帮助做成HTML发布到nishare，返回分享地址！
